# Tweeter Project

This is an AJAX-based twitter clone, it operates on a single page and makes use of jQuery as well as HTML. The project also uses CSS style templates to format the webpage design.

### Dependencies

- Express
- Node 5.10.x or above
- Body-parser
- Timeago.js
